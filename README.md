# GPT Text Generator

## Overview

This project generates meaningful text from a user prompt using a pre-trained DistilGPT2 model from the Hugging Face Transformers library.

## Technologies Used

* Python
* Hugging Face Transformers
* DistilGPT2
* Gradio
* Google Colab

## Features

* Accepts user prompts
* Generates AI-powered text
* Interactive web interface
* Lightweight and easy to run

## How to Run

Install dependencies:

pip install -r requirements.txt

Run:

python app.py

## Example

Input:
Artificial Intelligence is transforming education because

Output:
Artificial Intelligence is transforming education because it enables personalized learning experiences and improves accessibility to knowledge.
